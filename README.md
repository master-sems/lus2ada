## Getting started

To try this on your machine you need scala to be installed and the `scalac` command.
Run this once to make the debug.sh file executable :

```
chmod a+x debug.sh testPrinter.sh
```

To run just do this :

```
./debug.sh
```

To test the printer do this :

```
./testPrinter.sh
```
